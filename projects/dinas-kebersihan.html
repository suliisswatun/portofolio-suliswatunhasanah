Dokumentasi Desain

Proyek ini merupakan pengembangan website prototype sistem monitoring kendaraan pengangkut sampah berbasis IoT dan GPS smartphone untuk meningkatkan efisiensi operasional. Platform ini mampu mendeteksi potensi bahaya di lapangan, seperti kondisi sopir mengantuk, serta menyediakan pelacakan rute secara real-time dan rekomendasi jalur terdekat berdasarkan titik pembuangan sampah di wilayah Indramayu. Fitur monitoring kamera telah berhasil diimplementasikan secara real-time dan saat ini sistem masih dalam tahap pengujian sebelum implementasi operasional penuh.

Fitur Utama

  • Absensi dan manajemen data driver
  • Monitoring kendaraan menggunakan ESP32-CAM (real-time camera streaming)
  • Pelacakan posisi kendaraan berbasis GPS smartphone (real-time tracking)
  • Monitoring konsumsi bahan bakar berdasarkan estimasi jarak tempuh dan durasi perjalanan
  • Informasi detail kendaraan dan status kondisi operasional

Peran Saya

Dalam proyek ini, saya bertanggung jawab dalam pengembangan frontend dan backend dengan fokus pada pengembangan modul user (driver) dan admin. Pada sisi user, saya mengimplementasikan sistem registrasi serta logika absensi terintegrasi, di mana ketika driver melakukan absensi hadir, sistem monitoring akan aktif secara otomatis, dan akan nonaktif kembali saat driver menyelesaikan pekerjaan dan fitur rekomendasi rute perjalanan. Pada sisi admin, saya mengembangkan fitur autentikasi (register, login, lupa password, dan reset password) yang terintegrasi dengan Fonnte untuk memberikan kode unik yang digunakan pada reset password, serta dashboard admin, halaman monitoring, manajemen data driver, pengelolaan profil, monitoring bahan bakar, pelacakan posisi driver berbasis GPS.
