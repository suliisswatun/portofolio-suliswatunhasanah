WEBSITE SISTEM MANAJEMEN INFORMASI RUMAH MAKAN SEAFOOD DAPUR IBU SONIYA

Dokumentasi Desain

Proyek ini merupakan website sistem manajemen informasi dan pemesanan makanan untuk Rumah Makan Seafood Dapur Ibu Soniya yang bertujuan membantu UMKM dalam pengelolaan menu, transaksi pemesanan, dan pencatatan laporan keuangan secara terstruktur dan efisien. Source Code di GitHub

Fitur Utama

  • Manajemen data menu makanan
  • Sistem pencatatan laporan keuangan
  • Sistem pemesanan makanan berbasis web
  • Antarmuka pengguna yang sederhana dan user-friendly

Peran Saya

Dalam pembuatan proyek ini, saya bertanggung jawab sebagai ketua kelompok. Selain mengoordinasikan jalannya pengembangan proyek, saya juga berperan sebagai pengembang web. Tanggung jawab utama saya meliputi perancangan antarmuka pengguna (UI) serta pengembangan halaman admin, yang mencakup pembuatan dashboard admin yang bersifat dinamis dan real-time, pengelolaan data menu (tambah, ubah, dan hapus menu), pengelolaan antrian pesanan, serta pengolahan data keuangan. Selain itu, saya turut mengembangkan halaman pengguna dengan menambahkan informasi profil sebagai bagian dari personalisasi sistem dan tambah jumlah pesanan yang otomatis terkalkulasi pada form pemesanan.
